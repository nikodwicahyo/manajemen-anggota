HOW TO START WEBSITE

Jalankan server backend:
1. Buka XAMPP, kemudian Start Apache dan MySQL
2. buka vscode dan masuk folder backend, buka terminal ketik 'cd backend'
3. install dependensi ketik 'npm install'
3. jalankan server ketik 'nodemon index'

Jalankan Frontend:
1. Buka vscode dan masuk folder frontend, buka terminal ketik 'cd frontend'
2. install dependensi ketik 'npm install'
3. jalankan server ketik 'npm start'
