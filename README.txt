HOW TO START WEBSITE

Jalankan server backend:
1. Buka XAMPP, kemudian Start Apache dan MySQL
2. buka vscode dan buka terminal ketik 'cd backend'
3. jalankan server ketik 'nodemon index'

Jalankan Frontend:
1. Buka vscode dan buka terminal ketik 'cd frontend'
2. jalankan server ketik 'npm start'
